Install Anaconda version 23.7.4 + with python 3.11

Then open anaconda command promet and do the following installations



pip install opencv-python
pip install mss
pip install "paddleocr>=2.0.1" --upgrade PyMuPDF==1.21.1
pip install numpy==1.23.5
pip install paddlepaddle


Run python main.py and follor the video instructions for using the app
Result images will be saved into results folder in current dir.
